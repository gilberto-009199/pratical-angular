Formatar textos

Texto Original:

{{ 'eSsE e O TeXtO OrIgInAl' }}

Texto uppercase pipe:

{{ 'eSsE e O TeXtO OrIgInAl' | uppercase }}

Texto lowercase pipe:

{{ 'eSsE e O TeXtO OrIgInAl' | lowercase }}

Texto titlecase pipe:

{{ 'eSsE e O TeXtO OrIgInAl' | titlecase }}

Formatar Datas

Data Hora original:

{{ hoje }}

Data Hora com date:

{{ hoje | date }}

Data Hora com date:'dd/MM/yyyy':

{{ hoje | date:'dd/MM/yyyy' }}

Formatar Numeros e Moedas

Numero original:

{{ numero }}

Numero com number:'1.1-1':

{{ numero | number:'1.1-1' }}

Numero com number:'1.1-2':

{{ numero | number:'1.1-2' }}

Numero com currency:'BR':

{{ numero | currency:'BRL' }}

Numero com currency:'BR':'code':

{{ numero | currency:'BRL':'code' }}

Numero com currency:'BRL':'symbol':'1.2-2':

{{ numero | currency:'BRL':'symbol':'1.2-2' }}
