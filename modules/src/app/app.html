Modules:

Os modules como vc bme se elmbra são apra segmentar os recursos

Eles possuem 4 carascteristicas:

  • declarations: É onde você lista os componentes, diretivas e pipes que pertencem exclusivamente a este módulo.
  • imports: É onde você traz outros módulos cujas funcionalidades você deseja utilizar (ex: CommonModule para usar diretivas como o *ngIf).
  • providers: É onde você declara os serviços que ficarão disponíveis para os componentes deste módulo.
  • bootstrap: É o "ponto de partida". Define qual componente principal será carregado primeiro quando o módulo for iniciado (geralmente usado apenas no módulo raiz, o AppModule).

Exemplo:

Vamos criar um exemplo

Baseado nos setores de um empresa, Vendas e TI

$  ng g m vendas
  CREATE src/app/vendas/vendas-module.ts (192 bytes)

$  ng g m ti
  CREATE src/app/ti/ti-module.ts (188 bytes)

$  ng g m logistica
  CREATE src/app/logistica/logistica-module.ts (195 bytes)
